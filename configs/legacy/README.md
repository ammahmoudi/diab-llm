# Legacy Configuration Files

This directory contains old manual configuration files from early development.

## Status: OBSOLETE

These files are kept for historical reference only. **Do not use them.**

## Modern Workflow

All configurations are now auto-generated by modern config generators:

### Time-LLM Configs
```bash
python scripts/time_llm/config_generator_time_llm_unified.py \
    --mode train_inference \
    --llm_models BERT \
    --patients 570,584 \
    --seeds 42
```

### Chronos Configs
```bash
python scripts/chronos/config_generator.py \
    --mode train \
    --patients 570,584 \
    --models amazon/chronos-t5-base
```

### Distillation (Full Pipeline)
```bash
bash scripts/pipelines/distill_pipeline.sh \
    --teacher bert-base-uncased \
    --student prajjwal1/bert-tiny
```
